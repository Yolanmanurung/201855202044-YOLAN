Pengertian HTML

HypeText Markup Languange (HTML) merupakan suatu bahasa markah yang digunakan untuk membuat halaman web, menampilkan berbagai informasi di dalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolahan kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perintah-perintah HTML. HTML merupakan sebuah standar internet yang digunakan secara luas untuk menampilkan halaman web. Pada saat HTML merupakan sebuah standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Consortium (W3C). HTML dibuat oleh kolaborasi caillau TIM yang diketuai oleh Berners-lee robert ketika bekerja di CERN pada tahun 1989.
